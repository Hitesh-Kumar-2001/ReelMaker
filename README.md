# ReelMaker

A simple reel generator using Python and character assets.

## Usage

1. Configure character and prompt files.
2. Run `reelMaker.py` or the appropriate script in `scripts/main/`.
3. Keep video/audio assets out of version control by placing them in `bgVideo/`, `tempAudioFiles/`, or other ignored folders.

## Included folders

- `Characters/` — character assets and folders
- `scripts/` — application logic
- `posts/` — generated or scripted post content

## Notes

- `requirements.txt` lists Python dependencies.
- `.gitignore` excludes media and temporary files only.
